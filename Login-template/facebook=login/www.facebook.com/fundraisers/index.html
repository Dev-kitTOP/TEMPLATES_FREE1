triangle-right
¿Quién puede crear una recaudación de fondos?
En este momento, solo las personas de algunos países pueden crear recaudaciones de fondos en Facebook. Haz clic para ver el artículo con la lista completa de países.
Ver artículo completo
triangle-right
¿Cómo reciben donaciones las organizaciones sin fines de lucro?
Si creas una recaudación de fondos para una organización sin fines de lucro, esta recibirá las donaciones directamente de Facebook, de Network for Good o de PayPal Giving Fund.
Ver artículo completo
triangle-right
¿Qué impuestos se aplican?
Las donaciones a organizaciones sin fines de lucro de tipo 501(c)(3) de los EE. UU. suelen ser deducibles de impuestos, mientras que las donaciones a organizaciones benéficas que se encuentran fuera de los EE. UU. pueden estar sujetas a créditos o deducciones fiscales.
Consulta a un asesor para informarte si las donaciones son deducibles de impuestos.
Ver artículo completo
triangle-right
¿Qué cargos se aplican?
Facebook no cobra comisiones por las donaciones a organizaciones sin fines de lucro. Para obtener más información, lee el artículo completo.
Ver artículo completo
